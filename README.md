# MasterThesis_2026
This repository contains scripts written in R and Python used for the analyses performed for my master's thesis (May 2026). 

**Folder breakdown**

- Contact Maps: Python sripts for visualization of genome-wide Hi-C contact maps for the three species. 

- AB Compartments: Python scripts for A/B compartment calling in _A. rouxii_ and _M. flavus_. Using tutorial from cooltools. 

- TADs: Python scripts for TAD calling in _A. rouxii_ and _M. flavus_ using insulation score. Based on cooltools tutorial.

- Chromatin Loops: Python scripts for visualization of chromatin loops in _A. rouxii_ and _M. flavus_. 

- Additional Analyses: Python and R scripts for additional analyses, including ONT, PacBio, Hi-C coverage analysis, and analysis of multi-mapping reads.
